概述

简述在Hexo撰写一篇文章,以及写文章可能会用到的参数指令.

基础语法

标题

#放在标题前面，中间有个空格
与Html中的H1标签用法类似，不过后者是以数字大小决定内容大小，前者则是数量决定文字大小
最多可以支持6个

列表

有序列表:1. 2. 3.
无序列表:*+- 文本,符号与文本中间插入空格

  • 效果，三者都是一样的

代码文本

使用 ``` 代码语言
中间键入代码
结束时最后起一行```即可

if(imok)
   hollow world!

超链接

[文本](链接)
百度

插入图片

插入图片的方法:!图片

强调文本

在字两边加一个*为 斜体
在字两边加两个**为 粗体
在字两边加三个***为 粗斜体
不要有空格

分割线

连续三个下划线或减号

特殊符号插入

\斜杠一个再特殊符号前面即可

页内跳转

在要跳转到的位置<span id = “jump”>目标点\，然后使用[出发点](#jump)将你好设置为一单击即跳转到WORLD所在位置的效果
点一下我

居中居右居左

居中<center>文本</center>
居右<p align=”right”>文本</p>
居左<p align=”left”>文本</p>

换行

</br>或两个以上空格

删除线

使用 ~~ 将要删除文字夹起来。表示删除

字体样式

<font color=颜色编码 size=大小 face=”字体样式”>文本</font>
黑体字
color=#0099ff size=7 face=”黑体”
color=#00ffff
color=red

Size：支持 1 - 7 ，默认值 3
color：十六进制颜色值

待办事宜

横杠+空格+方括号+空格+文本 - [ ]方括号中大写的X来区分是否完成。
要注意空格空行。
